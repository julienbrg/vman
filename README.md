# VMAN

On-chain Version Manager.

## Install

```shell
npm i
```

## Test

```shell
npx hardhat test
```

## Deploy

Create a `.env` file at the root directory with your own keys.

Deploy to Aurora Testnet:

```shell
npx hardhat run scripts/deploy.ts --network aurora_testnet
```
