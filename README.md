# VMAN

On-chain Version Manager.

## Install

```shell
npm i
```

## Test

Create a `.env` file at the root directory with your own keys, then:

```shell
npx hardhat test
```

## Use

Deploy to Goerli:

```shell
npx hardhat run scripts/deploy.ts --network goerli
```

Update:

```shell
npx hardhat run scripts/update.ts --network goerli
```
